## 2. 리뷰 동영상 로컬재생 확장프로그램 최적화

리뷰 페이지 동영상 재생 시 CDN 트래픽을 완전히 제거한 최적화

### 문제
- 로컬 재생 개발 완료 상태에서도 매번 페이지 열 때마다 ~50kb CDN 트래픽 발생
- 원인: 브라우저가 `video src`를 파싱하는 즉시 원본 영상 다운로드 → 이후 JS로 교체하는 구조적 한계

### 해결 방안
- 브라우저가 원본 영상을 다운로드하기 전에 영상 주소를 선제적으로 제거
- 로컬 파일 존재 여부 확인 후: 있으면 로컬 재생, 없으면 원본 주소 복원하여 CDN 재생

### 개선 효과

| 항목 | 개선 전 | 개선 후 |
|------|---------|---------|
| 로컬 영상 CDN 트래픽 | ~50kb/페이지 | **0 B** |
| 로컬 없는 영상 재생 | 정상 (CDN) | 정상 (CDN) |
| 반복 새로고침 시 | 매번 50kb | **계속 0** |

## 3. 리뷰 AI 검수 시스템 — 해시 비교 & 중복영상 검지

동일 회원이 서로 다른 리뷰에 같은 영상을 재사용하는 사례를 자동 탐지하는 기능. AI 분석 전 Phase 1에서 선제 처리하여 불필요한 API 호출 비용을 절감.

### 3.1 3단계 하이브리드 해시 비교 전략

속도와 정확도를 동시에 확보하기 위해 단계적 필터링 구조를 채택.

| 단계 | 방식 | 설명 | 속도 |
|------|------|------|------|
| 1단계 | 파일 크기 그룹핑 | 파일 크기가 다르면 즉시 다른 영상으로 판정 | 즉시 |
| 2단계 | Quick MD5 | 파일 앞 1MB 바이너리 해시 비교 → 100% 동일 파일 탐지 | OpenCV 불필요 |
| 3단계 | 프레임 퍼셉추얼 해시 | 10%/50%/90% 위치 3프레임 → 128×128 리사이즈 → average hash + phash Hamming distance 비교 | CPU 병렬 (ProcessPoolExecutor) |

- **1~2단계에서 중복이 확정되면 3단계 스킵** → 대부분의 중복을 OpenCV 없이 초고속 처리
- 3단계는 **같은 회원 내 1~2단계를 통과한 쌍만** 정밀 검사 → 전체 대상 중 일부만 OpenCV 연산

### 3.2 프레임 해시 유사도 판정 기준

| 항목 | 설정값 | 설명 |
|------|--------|------|
| 영상 길이 차이 허용 | 1초 | 길이 차이가 1초 초과 시 다른 영상 |
| Average Hash 임계값 | Hamming ≤ 5 | 평균 해시 거리 |
| Perceptual Hash 임계값 | Hamming ≤ 10 | 퍼셉추얼 해시 거리 |
| 프레임 샘플링 | 10%, 50%, 90% | 3개 지점 프레임 추출 |
| 리사이즈 | 128×128 | 비교를 위한 정규화 |

### 3.3 Union-Find 클러스터링

- 각 회원별로 모든 영상 쌍을 비교 후 **Union-Find** 자료구조로 중복 그룹 클러스터링
- A=B, B=C인 경우 A=C 간접 중복까지 자동 그룹화
- 각 클러스터에서 **대표 영상 1개**를 선정 (기본: 가장 빠른 리뷰)
- 대표가 아닌 영상은 `중복리뷰`로 자동 마킹 → 이후 AI 분석 스킵

### 3.4 성능 최적화

| 기능 | 설명 |
|------|------|
| 해시 캐시 (`_hash_cache.json`) | 한 번 계산한 바이너리/프레임 해시를 영구 캐시에 저장 → 재실행 시 스킵 |
| CPU 병렬 연산 | `ProcessPoolExecutor`로 CPU 코어 절반만큼 병렬 프레임 해시 계산 |
| 바이너리 선필터 | Quick MD5로 대부분의 완전 중복을 OpenCV 없이 제거 |
| 대상 최소화 | 같은 회원 2건 이상인 경우만 비교, 1건 회원은 스킵 |

### 3.5 중복 검지 결과 보고서

| 항목 | 설명 |
|------|------|
| 출력 경로 | `hash_report/YYYYMMDD_중복체크결과.xlsx` |
| 통계 정보 | 전체 대상, 바이너리 중복, 시각적 중복, 해시 실패 건수 |
| 색상 구분 | 빨강(바이너리 중복), 주황(시각적 중복), 초록(대표), 파랑(단독), 회색(실패) |
| 상세 정보 | 리뷰번호, 회원번호, 파일크기, MD5, 영상길이, 해시값, 그룹ID 포함 |
