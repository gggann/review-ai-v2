# 리뷰 AI 정식버전2

11번가 이커머스 플랫폼용 **AI 기반 리뷰 동영상 자동 심사 시스템**입니다.  
리뷰 동영상을 다운로드하고, OpenAI GPT-4o 모델을 활용하여 2단계 AI 평가(부적합 검출 + 10점 만점 품질 평가)를 수행하며, 최종 결과를 Excel 파일로 출력합니다.

## ✨ 주요 기능

### 🔍 3단계 하이브리드 중복 동영상 감지
동일 회원이 같은 동영상을 반복 사용하는 리뷰를 AI 분석 전에 자동으로 식별하여 불필요한 API 호출을 방지합니다.

| 단계 | 방식 | 설명 |
|------|------|------|
| 1단계 | 파일 크기 그룹화 | 파일 크기가 다르면 즉시 제외 (순간 완료) |
| 2단계 | Quick MD5 | 파일 앞 1MB 바이너리 MD5 해시 비교 (OpenCV 불필요, 초고속) |
| 3단계 | 프레임 해시 | 10%/50%/90% 위치 3프레임의 지각 해시(pHash) 비교 (OpenCV 필요, CPU 병렬 처리) |

Union-Find 자료구조를 활용하여 간접 중복(A=B, B=C → A=C)을 자동으로 클러스터링합니다.

### 🚫 부적합 동영상 검출 (1단계 평가)
AI 및 CV(컴퓨터 비전) 기반으로 부적합 리뷰 동영상을 5가지 항목으로 검출합니다.

| 검출 항목 | 설명 |
|-----------|------|
| `is_wrong_product` | 영상 속 상품이 실제 구매 상품과 다름 |
| `is_irrelevant` | 영상이 상품과 완전히 무관함 |
| `is_delivery_box_only` | 택배 상자만 표시됨 (v1.9 이후 pass 처리) |
| `is_screen_recapture` | 화면 재촬영 (모아레 패턴, 암 테두리, 화면 중첩 등 CV 검출 포함) |
| `is_unverifiable` | 상품을 전혀 식별할 수 없음 |

**보수적 판단 원칙**: 의심스러운 경우 모두 정상(false)으로 판정하여 오검출을 최소화합니다.

### 📊 10점 만점 품질 평가 (2단계 평가)
3가지 독립 차원으로 리뷰 동영상의 품질을 평가합니다.

| 평가 차원 | 배점 | 설명 |
|-----------|------|------|
| **품질 (Quality)** | 0~2점 | 해상도, 화질, 선명도 |
| **가시성 (Visibility)** | 0~4점 | 상품 노출 정도 (전경+클로즈업+사용 장면) |
| **행동성 (Action)** | 0~4점 | 360도 회전, 사용 시연, 비교 등 적극적 표현 |
| **총점** | **0~10점** | 세 항목의 합산 |

카테고리별 특수 규칙(신선식품, 가공식품, 의류패션, 화장품뷰티, 생활주방, 전자가전, 잡화/가방/신발 등)이 자동으로 적용되며, 해당 카테고리의 최저 행동성 점수를 보장합니다.

### 🏷️ 적립금 지급 점수 자동 산정
AI 평가 결과를 바탕으로 적립금 지급 점수를 자동 계산합니다.

| 조건 | 지급 점수 |
|------|----------|
| 부적합 (0점) | **0점** |
| 총점 ≥ 7 또는 (총점 ≥ 5 그리고 행동성 ≥ 2) | **5점** |
| 그 외 | **2점** |

### 💾 체크포인트 & 이어서 진행
- 100건마다 자동으로 `_checkpoint.json`에 진행 상태를 저장합니다.
- 프로그램이 중단되더라도 다음 실행 시 이어서 처리할 수 있습니다.
- 완료 후 체크포인트 파일은 자동 삭제됩니다.

### 📝 Excel 결과 리포트
- 평가 결과를 색상으로 구분하여 직관적으로 표시합니다.
  - 🟢 초록: 고점 (5점 지급)
  - 🟡 노랑: 중간 (2점 지급)
  - 🟠 금색: 저점
  - 🔴 빨강: 부적합 (0점)
- 중복 동영상 감지 결과는 별도 Excel 리포트(`hash_report/`)로 출력됩니다.

## 🏗️ 시스템 아키텍처

```
Excel 입력 → main.py (전체 흐름 제어)
  │
  ├─ Phase 1: hash_utils.py (하이브리드 중복 감지)
  │    ├─ 파일 크기 그룹화
  │    ├─ Quick MD5 (1MB)
  │    └─ 프레임 지각 해시 (OpenCV 병렬)
  │    → 중복 리뷰 건너뛰기 (API 호출 없음)
  │
  ├─ Phase 2: 개별 AI 평가
  │    ├─ video_utils.py (다운로드 + 프레임 추출)
  │    ├─ zero_point.py (부적합 검출)
  │    │    ├─ CV 화면 재촬영 검출 (모아레, 암 테두리, 상태바, 화면 중첩)
  │    │    └─ AI 5항목 부적합 판정
  │    └─ five_point.py (10점 만점 평가)
  │         └─ AI 품질/가시성/행동성 평가
  │              + 카테고리 특수 규칙 보정
  │
  ├─ 정렬_및_레이아웃_처리.py (정렬 + 포맷팅)
  └─ Excel 출력 (색상 표시 + 적립금 지급 점수)
```

## 🔄 처리 흐름 상세

### Step 1: Excel 데이터 읽기
- 리뷰 동영상 데이터가 포함된 Excel 파일(`리뷰동영상_*.xlsx`)을 자동으로 탐색합니다.
- `.xls` 형식은 자동으로 `.xlsx`로 변환됩니다.
- A열 날짜를 기준으로 출력 경로를 자동 결정합니다.

### Step 2: 동영상 다운로드 및 프레임 추출
- 각 리뷰의 동영상을 로컬 경로에서 로드합니다 (LAN HTTP 또는 로컬 파일).
- 동영상 길이에 따라 동적으로 프레임 수를 결정합니다:
  - 짧은 동영상 (< 15초): 8프레임
  - 일반 동영상 (15~60초): 10프레임
  - 긴 동영상 (≥ 60초): 15프레임
- 가중 샘플링 (전반부 40% + 후반부 50% + 마지막 5%)으로 영상 전체를 골고루 반영합니다.
- 정지 이미지 감지, 흑색 화면 감지, MSE 기반 유사 프레임 제거를 수행합니다.

### Step 3: 부적합 동영상 필터링 (1단계 평가)
- **CV 검출**: 모아레 패턴(FFT), 균일 암 테두리, 상태바 패턴, 화면 중첩(중첩 사각형) 감지
- **AI 검출**: GPT-4o가 상품 정보, CV 사전 검사 결과, 상품 이미지, 동영상 프레임 그리드를 분석하여 5가지 부적합 항목 판정
- **보수적 원칙**: 의심스러운 경우 모두 정상으로 판정

### Step 4: 10점 만품 품질 평가 (2단계 평가)
- 부적합이 아닌 동영상에 대해 3차원 평가 수행
- 프레임 그리드 이미지를 GPT-4o에 전달하여 시각적 분석
- JSON 규칙 파일(`five_point_rules.json`)에서 동적으로 프롬프트를 조립
- 카테고리별 특수 규칙이 자동 감지되어 최저 점수 보장

### Step 5: 적립금 지급 점수 계산
- AI 평가 점수를 기반으로 0/2/5점 적립금 지급 점수를 산정

### Step 6: 결과 Excel 저장
- 모든 평가 결과를 원본 Excel 파일에 추가 컬럼으로 저장
- 100건 단위로 자동 저장 (체크포인트)
- 정렬 및 색상 포맷팅 적용

## 📏 평가 기준

### 10점 만점 평가 상세

#### 품질 (Quality) — 0~2점
| 점수 | 기준 |
|------|------|
| 2점 | 우수 — HD급 이상 해상도, 선명하고 깨끗한 화질 |
| 1점 | 보통 — 감상에 지장이 없는 수준 |
| 0점 | 불량 — 상품 식별이 불가능할 정도로 흐릿함 |

#### 가시성 (Visibility) — 0~4점
| 점수 | 기준 |
|------|------|
| 4점 | 극우수 — 전경+클로즈업+사용 장면이 모두 포함 |
| 3점 | 우수 — 상품이 명확하게 보이며 다양한 각도 포함 |
| 2점 | 보통 — 상품 식별 가능, 일부 각도만 표시 |
| 1점 | 부족 — 상품이 잘 보이지 않음 |
| 0점 | 식별 불가 — 상품이 화면에 나타나지 않음 |

#### 행동성 (Action) — 0~4점
| 점수 | 기준 |
|------|------|
| 4점 | 극풍부 — 360도 회전+사용 시연+비교 등 3가지 이상 포함 |
| 3점 | 풍부 — 다양한 시연 장면 포함 |
| 2점 | 기본 — 기본적인 상품 소개 |
| 1점 | 최소 — 거의 움직임 없음 |
| 0점 | 없음 — 정지 화면과 동일 |

### 카테고리별 특수 규칙
특정 상품 카테고리는 최저 행동성 점수를 보장합니다:

| 카테고리 | 최저 행동성 점수 | 특이사항 |
|----------|-----------------|----------|
| 신선식품 | 2점 | 포장 개봉, 단면 촬영 등 기본 행동 필요 |
| 가공식품 | 2점 | 제품 상세 표시, 조리/시식 장면 권장 |
| 의류/패션 | 2점 | 착용 장면 필수 |
| 화장품/뷰티 | 2점 | 도포/적용 장면 필수 |
| 생활/주방 | 2점 | 사용 시연 필수 |
| 전자/가전 | 2점 | 작동/기능 시연 필수 |
| 잡화/가방/신발 | 2점 | 다각도 표시 필수 |

## ⚙️ 설정 가이드

### config.py 주요 설정

| 설정 항목 | 기본값 | 설명 |
|-----------|--------|------|
| `MAX_CONCURRENT_TASKS` | 10 | 최대 동시 처리 건수 |
| `TEST_MODE` | `False` | 테스트 모드 활성화 |
| `TEST_LIMIT` | 30 | 테스트 모드 시 처리 건수 |
| `DUPLICATE_HASH_THRESHOLD` | 5 | 중복 감지 Hamming 거리 임계값 |
| `DUPLICATE_DURATION_MARGIN` | 1 | 동영상 길이 허용 오차 (초) |
| `QUICK_HASH_BYTES` | 1MB | Quick MD5 읽기량 |
| `FRAME_SAMPLE_POSITIONS` | [0.1, 0.5, 0.9] | 프레임 샘플링 위치 |

### 규칙 파일 커스터마이징
- **`five_point_rules.json`**: 10점 만점 평가 기준, 카테고리별 규칙, 강제 규칙 수정
- **`zero_point_rules.json`**: 부적합 검출 항목, 보수적 판단 기준, CV 검출 임계값 수정
